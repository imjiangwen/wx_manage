# wx_manage

#### 介绍
微信后台管理页面
#### 注意
自己写接口，接口名字和参数详见各个页面的请求。<br/>
/src/utils/httpRequest.js修改接口基础地址和请求拦截等
#### 运行
npm install<br/>
npm run serve
#### 打包
npm run build
